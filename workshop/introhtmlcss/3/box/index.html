CSS Box Model

Margin
Border
Padding
Content

Basic elements

  • HTML elements conceived as boxes.
  • Margin: Clear area around border of box.
  • Border: Boundary around box.
  • Padding: Area between border and content. Takes background color.
  • Content: Where text and images appear.

Scaling

  • Static: Using hard coded pixel numbers.
  • Fluid: Using auto or inherited values, or percentages.
  • Adjusting: Set minimum and maximum values for elements that inherit size.
  • Overflow: Determines whether content that doesn't fit in the specified size is displayed. Can be 
    hidden, visible, scroll, inherit, auto

Aesthetics

Background

  • Images: Can be fixed or move with element scroll. Like HTML background.
  • Color
  • Repeat: x-axis or y-axis.
  • Set position: Set image from top or bottom, left or right.
  •  div.examples {
	background-color: red;
	background-image: url(cat-ears.jpg);
	background-repeat:repeat-x;
	}

Other properties

  • Rounded corners: Use radius to give box rounded corners.
  • Opacity: Make elements show other elements placed behind them.
  • Drop shadow: Nice way to pop elements out.
  •  div {
	background-color:blue;
	box-shadow: 10px 10px 5px #888888;
	border-top-left-radius: 30px;	
	opacity:0.5;
	}
