Grid Systems

  • In design, the "format" is the margins of the page that hold a design.
  • A grid system is the measurements that are used to align and size objects within the format.
  • There is no wrong way to set up a grid, but there are types of grids that are commonly used.
  • Grids are often created in Photoshop or Illustrator but are actually easier to make with CSS and HTML because we are limited to numbers.
