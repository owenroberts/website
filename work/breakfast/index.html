Based on the story line bird breakfast from my comic lines, and the animated version, breakfast.owen.cool is a multiplayer, 3D, web based interactive environment.

This project is a both a work in progress and a proof of concept for a larger project I've been working on. It combines the 3D animations I started working on with the hot dogs trilogy and the networked stuff I've been doing with some other projects. It's multiplayer, so bring some friends along if you want to try it out. However, if you don't have friends, the server will add some fake ones for you.
